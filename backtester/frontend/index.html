How to write a strategy 
class Strategy:
    def setup(self, ctx):
        ctx.params["key"] = 1.0   # optional

    def on_bar(self, snapshot, ctx):
        # Read market: snapshot.spot, snapshot.atm_strike,
        # snapshot.option(strike, "CE"), snapshot.by_offset(n, "PE")

        if ctx.position:
            return ctx.exit(reason="…")   # or ctx.hold()

        if some_condition_fails:
            return ctx.skip(reason="exact reason here")

        return ctx.enter(side="CE", strike="ATM", direction="BUY",
                         qty=1, reason="why entering")

You must call exactly one of: enter, exit, skip, or hold each bar.
